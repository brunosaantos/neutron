  • <img src="{{img.src}}" alt="{{img.alt}}" {{#if styleModifier}}class="{{styleModifier}}"{{/if}}>

  • A simple image element.

    • img.src (String)
      Supported values: A valid image address.
    • img.alt (String)
      Supported values: A description of the image. Ideal for accessibility and SEO.
    • styleModifier (String)
      Supported values: If necessary to add classes to image atom, add it using style modifier.
