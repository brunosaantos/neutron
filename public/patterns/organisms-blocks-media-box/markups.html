    • boxTitle (String)
      Supported values: Text to be used as main title.
    • collection (Array)
      Supported values: An object containing image atom object, headline text and description text.
    • collection.image (Object)
      Supported values: Image supports all properties of atoms/media/image atom.
    • collection.headline (String)
      Supported values: Headline text
    • collection.text (String)
      Supported values: Description text
