  • Thumbnail molecule to print boxes with image, headline, text and button.

    • thumbnails (Array)
      Supported values: An array containing headline atom object, text and button object.
    • thubnails.headline (Object)
      Supported values: The headline object supports the text property containing the headline content.
    • thubnails.text (String)
      Supported values: The text to be added to thumbnail molecule.
    • thubnails.button (Object)
      Supported values: The button object has the same properties supported by atoms/text/link.
